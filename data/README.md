# Data README

All data in this folder is synthetic and reproducible. It is generated by `scripts/generate_bi_artifact.py` with a fixed random seed.

The data is modeled on a natural and specialty grocery distribution analytics workflow:

- Customer and banner reference data.
- Natural, organic, specialty, fresh, wellness, and frozen category structures.
- SKU-level attributes similar to syndicated product intelligence dimensions.
- Weekly POS dollars, units, ACV distribution, velocity, syndicated share, competitor share, retailer scan units, and panel repeat rate.
- Distributor shipment, case-fill, inventory, purchase-order, service-level, and spoilage-risk signals.
- Promotion calendars with planned lift, actual lift, merchandising support, compliance, and funding rate.
- Data quality checks for source freshness, scan reconciliation, item hierarchy, promotion joins, stakeholder requirement clarity, and Power BI refresh SLA.
- Stakeholder requests, Power BI assets, and DAX-style semantic measures.

The data does not represent any real distributor, customer, supplier, category performance, or retailer performance.
