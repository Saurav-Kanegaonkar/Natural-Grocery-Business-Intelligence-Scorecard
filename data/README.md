# Data Sources

Synthetic source-style data for natural grocery distribution BI analytics.

- `entities.csv`: 36 customer category records.
- `daily_metrics.csv`: 5,040 customer category-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
